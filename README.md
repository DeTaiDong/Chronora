# Chronora

Chronora 是一个围绕八字排盘做的小型 WebApp。

项目主要用来整理出生信息、生成四柱命盘，并展示五行、十神、神煞、真太阳时等基础内容。用户也可以补充想看的问题，再进入命盘解读页面。

## 功能

- 八字四柱排盘
- 出生地搜索与真太阳时修正
- 五行分布、藏干、十神、神煞展示
- 常用资料保存，方便下次快速填入
- 基于命盘和补充信息的一次性解读

## 技术栈

- Next.js
- React
- TypeScript
- Tailwind CSS
- lunar-typescript

## 本地运行

```bash
npm install
npm run dev
```

如需使用解读功能，需要参考 `.env.example` 配置接口密钥。

## 说明

命理内容只作为传统文化和娱乐参考，不构成医疗、法律、投资或人生重大决策建议。

不同流派对部分神煞和取法会有差异，当前实现仍会继续校正。
